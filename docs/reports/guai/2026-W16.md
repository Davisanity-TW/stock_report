# EP653 / 4月15日

## 1. 市場概況

**多頭行情極其強勁：**

目前台美股市場的走勢非常好，甚至好到讓人開始懷疑人生 。美股近期已展現「大復活」的姿態，而台股則由各路飆股持續向前衝刺 。投資者應好好把握這波強勢多頭，確保在行情來臨時能精準咬住獲利標的 。

**台股指數的「台積電形狀」：**

台股指數目前的走勢幾乎就是台積電股價的形狀 。若以市場共識預估台積電明年 EPS 可達 130 至 140 元，並給予 20 倍本益比計算，股價來到 2800 元並非誇張數字 。這代表台股指數向上發展的空間依然具備相當大的潛力 。

**市場進入「搶錢」階段：**

主委觀察到目前是典型的「搶錢行情」，許多投資者的共同點是手上能用的資金太少，但想買的東西太多 。這導致資金在強勢族群間快速輪動，甚至出現為了買進更強的標的而汰換掉漲得慢的股票之現象 。這種動能驅動的環境要求投資者具備極高的操作彈性 。

**CPU 需求引發供應鏈甩動：**

市場正驚覺 CPU 的需求與 GPU、ASIC 一樣呈現極度緊缺的狀態 。為了運行 Agentic AI（AI 代理人），大量的 CPU 算力是不可或缺的，這導致整個供應鏈再次被大幅甩動 。當大家開始瘋狂搶購通用型或客製化 CPU 時，將帶動相關零組件的結構性上漲 。

**被動元件出現「缺貨大潮」徵兆：**

國際大廠太陽誘電已宣布全系列產品線漲價，預告了 MLCC、電容與電桿的緊缺趨勢 。特別是鋁電容（SPCAP）目前呈現極度供不應求的情況，部分 ODM 廠商已被要求漲價 。這波走勢有可能演變成類似 2017 至 2018 年的被動元件超級大行情 。

**Google 家族題材成為熱點：**

圍繞 Google TPU 供應鏈的討論正盛，包含 IP、TGV（玻璃通孔）以及 IPD（整合被動元件）等相關技術 。這些與 Google 晶片設計相關的標的，往往在新聞獲得驗證前就已領先起跑 。只要 Google 的資本支出維持穩定，這個家族的表現就值得密切注意 。

## 2. 關注的股票

**聯發科 (2454)：**

聯發科與 Google 合作開發 V8 (Zebra Fish) 與 V9 (H Fish) 兩顆 TPU 晶片，預期投片量極高 。這兩顆晶片有望在三至四年內為聯發科貢獻超過一半的 EPS，使其在 AI 領域的進展顯著領先競爭對手高通 。這種晶片設計代工模式，對聯發科的獲利結構將產生質變 。

**Marvell (MVEL)：**

Marvell 被證實成為 Google 在晶片設計上的新合作夥伴，目前已知有兩顆晶片開案 。其中一顆專注於推理（Inference），另一顆則與記憶體處理有關，旨在對標 NVIDIA 的 LPU 方案 。這不僅鞏固了 Marvell 的估值，也顯示其在高速傳輸與記憶體整合技術上的核心競爭力 。

**台積電 (2330)：**

儘管台積電股價已高，但以未來的 EPS 展望來看，在 AI 族群中仍不算貴 。目前 Google 開出的產能需求已遠超台積電能提供的負荷，這可能促使台積電進一步上修其財務展望 。作為全球先进製程的壟斷者，台積電依然是整個 AI 浪潮中最紮實的核心 。

**CPU 族群 (Intel, ARM, AMD)：**

這些是主委目前美股的前五大持股，核心邏輯在於 Agentic AI 對處理器需求的重新定價 。隨著通用處理器需求回升，這些標的預期還有繼續走強的機會 。在多頭行情中，這些具備長線故事支持的「愛與信仰」標的較能持股穩定 。

**被動元件族群 (太陽誘電, 國巨)：**

隨著國際同業漲價，國巨（2327）的法說會展望備受市場期待 。投資者應關注整體族群（如村田、TDK、太陽誘電）是否同步創下歷史新高 。若整個族群整包啟動，則代表全球性的缺貨趨勢成形，具備極高的獲利空間 。

**啟碁 (6285)：**

啟碁在 SpaceX 的 Starlink 訂單中獲得顯著佔比，單一客戶訂單金額甚至佔其市值的一半 。這類衛星通訊標的雖然因記憶體缺料可能面臨出貨延遲，但訂單剛性需求依然存在 。雖然 EPS 貢獻會因毛利結構而有所收斂，但產業趨勢已在第一線獲得驗證 。

## 3. 其他財經消息及知識

**晶片設計代工模式差異：**

博通（Broadcom）採取全套包辦模式，從設計到 HBM 採購皆一手掌握，利潤最豐厚 。聯發科則採專業分工，負責 IO 與 memory buffer，讓 Google 保留更多核心技術與議價權 。這種模式對 Google 而言更具靈活性，也為聯發科帶來大量且穩定的訂單流 。

**記憶體整合是 AI 的一切：**

Marvell 高度強調 AI 運算的效率取決於 SRAM 與 HBM 的整合 。其為 Google 開發的 memory processing unit 就是為了消弭運算瓶頸，提升推理速度 。這顯示出未來 AI 晶片的競爭將不再僅是算力，而是存取效率的全面對決 。

**傑文斯悖論的持續應用：**

雖然技術進步可能降低單位硬體需求，但效率提升往往誘發更廣泛的應用需求，最終總產量反而增加 。這套邏輯解釋了為何 AI 效能提升後，對伺服器、記憶體與電力設備的需求反而呈現爆炸性增長 。投資者不應因小幅度的優化技術而擔憂硬體供應鏈被取代 。

**「定價錯誤」帶來的 Alpha：**

主動投資者的獲利來源在於識別市場暫時的定價錯誤 。主委強調自己具備高度彈性，當市場條件達成或趨勢轉向時，會毫不猶豫地調整策略 。這種不墨守成規、敢於「善變」的特質，是其在多變市場中生存的利器 。

**三分鐘熱度的正面思考：**

主委認為對新事物擁有一頭熱的好奇心並非缺點，而是探索世界的探險精神 。當你了解一個領域的運作邏輯（如清酒、紅酒文化）後不流連忘返，反而能讓你更客觀地看待市場 。這種博採眾長、迅速掌握核心的能力，也能轉化為對不同產業趨勢的敏銳洞察 。

**NordVPN 的多元實用性：**

在規劃旅遊時，利用 VPN 切換不同地區（如新世界與舊世界）可以發現機票與住宿的顯著價差 。此外，VPN 提供的網路加密保護能確保在公共 WiFi 環境下查詢行程與付款的安全 。對於遊戲玩家，透過換區甚至能解決電纜過慢或領取特定地區寶物的需求 。

## 4. Q&A

**育兒溝通與情緒管理：**

面對孩子在外鬧脾氣，主委建議將其帶離現場（如餐廳）以冷靜溝通 。父母應理解孩子的大腦發育尚未完全，吼叫只會讓孩子模仿負面情緒，甚至造成心理創傷 。多一點愛與耐心，在溝通中發現孩子哭鬧的可愛之處，是珍惜親子時光的關鍵 。

**趨勢判斷的標準答案：**

投資並沒有絕對的均線或天數準則，標榜完美的人多半是在演戲 。最簡單的方式是看「新高」，因為新高股沒有解套賣壓且趨勢明確 。投資者應建立自己的模組與方法，接受生命與交易中的不完美，而非追求電腦都能完成的標準公式 。

**機會成本的操作應用：**

被減碼的股票不一定是因為它跌，而是因為它「長太慢」 。當市場出現更有動能、肉更多的標的時，應果斷將資金提款並移往強勢處 。這種隨時評估資金更好去處的思維，能顯著優化投資組合的整體報酬率 。

**地緣政治的「鈍化」效應：**

對於中東衝突等事件，只要油價不突破前高，主委就將其視為影響力逐漸減弱的雜訊 。這種基於經驗法則的判斷，能讓投資者在美股出現大紅 K 等技術訊號時，具備持股底氣而不被嚇跑 。市場情緒轉變極快，維持清醒並避免被過度的 FUD 左右是操作難點 。

**戒菸與環境的重要性：**

戒菸需要多次嘗試並找出訣竅，尤其是要離開過往會想抽菸的環境 。主委分享「深呼吸法」對部分聽眾有效，因為吸菸的爽感部分來自於深呼吸的節奏 。定期進行健康檢查看到身體的變化，也能提供實質的動機來維持戒菸成果 。

**職場關係與人生選擇：**

雖然老師從小教導要定性，但掌握自己善變且好奇的個性也能在社會上過得不錯 。如果你的個性不適合高重複性、不能失誤的工作（如會計），就應尋找更適合發揮創意與彈性的職涯路徑 。了解自己並與之共處，遠比勉強自己符合社會既定模板更有價值 。
