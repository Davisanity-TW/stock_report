2026-04-15 ​

EP653 / 4月15日 ​

1. 市場概況 ​

多頭行情極其強勁：

目前台美股市場的走勢非常好，甚至好到讓人開始懷疑人生 。美股近期已展現「大復活」的姿態，而台股則由各路飆股持續向前衝刺 。投資者應好好把握這波強勢多頭，確保在行情來臨時能精準咬住獲利標的 。

台股指數的「台積電形狀」：

台股指數目前的走勢幾乎就是台積電股價的形狀 。若以市場共識預估台積電明年 EPS 可達 130 至 140 元，並給予 20 倍本益比計算，股價來到 2800 元並非誇張數字 。這代表台股指數向上發展的空間依然具備相當大的潛力 。

市場進入「搶錢」階段：

主委觀察到目前是典型的「搶錢行情」，許多投資者的共同點是手上能用的資金太少，但想買的東西太多 。這導致資金在強勢族群間快速輪動，甚至出現為了買進更強的標的而汰換掉漲得慢的股票之現象 。這種動能驅動的環境要求投資者具備極高的操作彈性 。

CPU 需求引發供應鏈甩動：

市場正驚覺 CPU 的需求與 GPU、ASIC 一樣呈現極度緊缺的狀態 。為了運行 Agentic AI（AI 代理人），大量的 CPU 算力是不可或缺的，這導致整個供應鏈再次被大幅甩動 。當大家開始瘋狂搶購通用型或客製化 CPU 時，將帶動相關零組件的結構性上漲 。

被動元件出現「缺貨大潮」徵兆：

國際大廠太陽誘電已宣布全系列產品線漲價，預告了 MLCC、電容與電桿的緊缺趨勢 。特別是鋁電容（SPCAP）目前呈現極度供不應求的情況，部分 ODM 廠商已被要求漲價 。這波走勢有可能演變成類似 2017 至 2018 年的被動元件超級大行情 。

Google 家族題材成為熱點：

圍繞 Google TPU 供應鏈的討論正盛，包含 IP、TGV（玻璃通孔）以及 IPD（整合被動元件）等相關技術 。這些與 Google 晶片設計相關的標的，往往在新聞獲得驗證前就已領先起跑 。只要 Google 的資本支出維持穩定，這個家族的表現就值得密切注意 。

2. 關注的股票 ​

聯發科 (2454)：

聯發科與 Google 合作開發 V8 (Zebra Fish) 與 V9 (H Fish) 兩顆 TPU 晶片，預期投片量極高 。這兩顆晶片有望在三至四年內為聯發科貢獻超過一半的 EPS，使其在 AI 領域的進展顯著領先競爭對手高通 。這種晶片設計代工模式，對聯發科的獲利結構將產生質變 。

Marvell (MVEL)：

Marvell 被證實成為 Google 在晶片設計上的新合作夥伴，目前已知有兩顆晶片開案 。其中一顆專注於推理（Inference），另一顆則與記憶體處理有關，旨在對標 NVIDIA 的 LPU 方案 。這不僅鞏固了 Marvell 的估值，也顯示其在高速傳輸與記憶體整合技術上的核心競爭力 。

台積電 (2330)：

儘管台積電股價已高，但以未來的 EPS 展望來看，在 AI 族群中仍不算貴 。目前 Google 開出的產能需求已遠超台積電能提供的負荷，這可能促使台積電進一步上修其財務展望 。作為全球先进製程的壟斷者，台積電依然是整個 AI 浪潮中最紮實的核心 。

CPU 族群 (Intel, ARM, AMD)：

這些是主委目前美股的前五大持股，核心邏輯在於 Agentic AI 對處理器需求的重新定價 。隨著通用處理器需求回升，這些標的預期還有繼續走強的機會 。在多頭行情中，這些具備長線故事支持的「愛與信仰」標的較能持股穩定 。

被動元件族群 (太陽誘電, 國巨)：

隨著國際同業漲價，國巨（2327）的法說會展望備受市場期待 。投資者應關注整體族群（如村田、TDK、太陽誘電）是否同步創下歷史新高 。若整個族群整包啟動，則代表全球性的缺貨趨勢成形，具備極高的獲利空間 。

啟碁 (6285)：

啟碁在 SpaceX 的 Starlink 訂單中獲得顯著佔比，單一客戶訂單金額甚至佔其市值的一半 。這類衛星通訊標的雖然因記憶體缺料可能面臨出貨延遲，但訂單剛性需求依然存在 。雖然 EPS 貢獻會因毛利結構而有所收斂，但產業趨勢已在第一線獲得驗證 。

3. 其他財經消息及知識 ​

晶片設計代工模式差異：

博通（Broadcom）採取全套包辦模式，從設計到 HBM 採購皆一手掌握，利潤最豐厚 。聯發科則採專業分工，負責 IO 與 memory buffer，讓 Google 保留更多核心技術與議價權 。這種模式對 Google 而言更具靈活性，也為聯發科帶來大量且穩定的訂單流 。

記憶體整合是 AI 的一切：

Marvell 高度強調 AI 運算的效率取決於 SRAM 與 HBM 的整合 。其為 Google 開發的 memory processing unit 就是為了消弭運算瓶頸，提升推理速度 。這顯示出未來 AI 晶片的競爭將不再僅是算力，而是存取效率的全面對決 。

傑文斯悖論的持續應用：

雖然技術進步可能降低單位硬體需求，但效率提升往往誘發更廣泛的應用需求，最終總產量反而增加 。這套邏輯解釋了為何 AI 效能提升後，對伺服器、記憶體與電力設備的需求反而呈現爆炸性增長 。投資者不應因小幅度的優化技術而擔憂硬體供應鏈被取代 。

「定價錯誤」帶來的 Alpha：

主動投資者的獲利來源在於識別市場暫時的定價錯誤 。主委強調自己具備高度彈性，當市場條件達成或趨勢轉向時，會毫不猶豫地調整策略 。這種不墨守成規、敢於「善變」的特質，是其在多變市場中生存的利器 。

三分鐘熱度的正面思考：

主委認為對新事物擁有一頭熱的好奇心並非缺點，而是探索世界的探險精神 。當你了解一個領域的運作邏輯（如清酒、紅酒文化）後不流連忘返，反而能讓你更客觀地看待市場 。這種博採眾長、迅速掌握核心的能力，也能轉化為對不同產業趨勢的敏銳洞察 。

NordVPN 的多元實用性：

在規劃旅遊時，利用 VPN 切換不同地區（如新世界與舊世界）可以發現機票與住宿的顯著價差 。此外，VPN 提供的網路加密保護能確保在公共 WiFi 環境下查詢行程與付款的安全 。對於遊戲玩家，透過換區甚至能解決電纜過慢或領取特定地區寶物的需求 。

4. Q&A ​

育兒溝通與情緒管理：

面對孩子在外鬧脾氣，主委建議將其帶離現場（如餐廳）以冷靜溝通 。父母應理解孩子的大腦發育尚未完全，吼叫只會讓孩子模仿負面情緒，甚至造成心理創傷 。多一點愛與耐心，在溝通中發現孩子哭鬧的可愛之處，是珍惜親子時光的關鍵 。

趨勢判斷的標準答案：

投資並沒有絕對的均線或天數準則，標榜完美的人多半是在演戲 。最簡單的方式是看「新高」，因為新高股沒有解套賣壓且趨勢明確 。投資者應建立自己的模組與方法，接受生命與交易中的不完美，而非追求電腦都能完成的標準公式 。

機會成本的操作應用：

被減碼的股票不一定是因為它跌，而是因為它「長太慢」 。當市場出現更有動能、肉更多的標的時，應果斷將資金提款並移往強勢處 。這種隨時評估資金更好去處的思維，能顯著優化投資組合的整體報酬率 。

地緣政治的「鈍化」效應：

對於中東衝突等事件，只要油價不突破前高，主委就將其視為影響力逐漸減弱的雜訊 。這種基於經驗法則的判斷，能讓投資者在美股出現大紅 K 等技術訊號時，具備持股底氣而不被嚇跑 。市場情緒轉變極快，維持清醒並避免被過度的 FUD 左右是操作難點 。

戒菸與環境的重要性：

戒菸需要多次嘗試並找出訣竅，尤其是要離開過往會想抽菸的環境 。主委分享「深呼吸法」對部分聽眾有效，因為吸菸的爽感部分來自於深呼吸的節奏 。定期進行健康檢查看到身體的變化，也能提供實質的動機來維持戒菸成果 。

職場關係與人生選擇：

雖然老師從小教導要定性，但掌握自己善變且好奇的個性也能在社會上過得不錯 。如果你的個性不適合高重複性、不能失誤的工作（如會計），就應尋找更適合發揮創意與彈性的職涯路徑 。了解自己並與之共處，遠比勉強自己符合社會既定模板更有價值 。

2026-04-18 ​

EP654 / 4月18日 ​

1. 市場概況 ​

美台股市同步展現極強動能：

目前美股的漲幅非常驚人，被形容為「直接一根針往天上頂」，顯示出強勁的噴發態勢 。台股同樣屬於非常強勢的格局，整體走勢與台積電的表現高度連動，形成所謂的「台積電形狀」 。主委預期這波上檔空間可能會超出多數人的想像，投資者應做好心理準備迎接更大的行情 。

多頭市場的操作思維：

在目前明顯的強勁多頭形態下，市場的邏輯已轉變為「拉回就是買點」，即便出現跳樓式的下殺也不應過度恐慌 。雖然市場上有人會預警「業力引爆」或崩盤，但因為前面拉出的紅 K 空間極大，這些修正通常會被視為上車機會 。在趨勢明朗的情況下，尋求各類題材介入並儘可能在行情好時多賺一點，是目前的最佳策略 。

台積電法說會與資本支出表態：

台積電在最近的展望中給出了非常好的指引，雖然資本支出並未如部分人預期地直接上修，但維持在原定的高標區間 。這種表態本身就是一種強烈的信心訊息，預示著公司對未來訂單的看法依然非常樂觀 。主委認為未來仍有機率看到台積電正式上修資本支出，這將是後續市場關注的驚喜點 。

場務與設備類股的補漲機會：

過往表現較為沈悶的場務與長物類股，在台積電法說會後反而開始出現表態，如漢唐 (2404) 在這段時間的表現相當出色 。主委分析這類股票目前屬於市場中相對便宜的區塊，雖然他曾因跑得慢而減碼，但其成長性依然穩健 。資金有時會從過熱的動能股流向這類具備價值且有實質業績支撐的標的 。

全球被動元件族群集體噴射：

全球範圍內的被動元件類股正出現集體大漲的現象，包含日本標的以及美國的 Vishay (VSH) 等公司都已射出 。雖然國巨 (2327) 在法說會上的態度相對保守神聖，但其股價其實已默默爬回到還原權息的新高位置 。這顯示國際市場正領先反映被動元件的復甦趨勢，台灣相關族群也有望隨後跟上 。

記憶體族群的台美股表現分歧：

記憶體報價雖然持續上漲，但台股相關標的目前的走勢明顯比美股疲弱，顯示市場資金可能已經過度擁擠 。當多數資金都已上車且缺乏新的強大催化劑時，股價就容易陷入橫盤，導致操作效率降低 。主委建議在這種情況下，資金可能會轉向其他下一個具備缺貨漲價故事的族群，如被動元件或 ABF 載板 。

2. 關注的股票 ​

台積電 (2330) 與其資本支出供應鏈：

台積電不僅是台股的定海神針，其資本支出的變動也直接影響了相關設備與場務廠商的命運 。雖然目前數字未上修，但市場已將其視為「拉回即買」的多頭核心，只要訂單沒有出問題，股價就會往合理位階靠攏 。投資者應持續追蹤其資本支出是否有驚喜的上修，這將帶動整包供應鏈的再次噴發 。

國巨 (2327) 與國際被動元件：

國巨作為台股被動元件龍頭，雖然在法說會中未大肆吹捧，但其目前的位階已處於還原權息後的新高 。國際村時代下，日本與美國的被動元件大廠如 Vishay、Eaton 等都已強勢大漲，預示產業可能有大事發生 。主委認為不一定要死守單一標的，若覺得美日股效率更高，也可往該方向配置 。

漢唐 (2404) 等場務與長物股：

漢唐近期在市場中表現強勁，且剛好挑在台積電法說會後表態，讓市場再次注意到場務標的的便宜特質 。雖然這類股票過往被認為是價值型而非動能型，但目前的資金輪動讓其重新獲得青睞 。只要半導體擴產趨勢不變，這類提供無塵室與相關基礎設施的公司便具備長期保障 。

Intel 與 AMD 在 CPU 領域的潛力：

CPU 的缺貨情況在未來下半年可能比記憶體更為嚴重，且目前 Intel 與 AMD 的漲幅相對落後，具備補漲空間 。Agentic AI 的發展高度依賴 CPU 的算力，這是一個目前市場尚未充分預期的「房間裡的大象」題材 。如果後續出現 CPU 買不到而引發的漲價潮，相關處理器大廠的獲利將會非常驚人 。

萬潤 (6187) 與弘塑 (3131) 等 CPO 測試設備：

隨著光通訊技術從傳統可插拔式轉向 CPO (共同封裝光學)，良率不穩定的特性讓測試設備變得極其重要 。主委提到自己長期持有且看好這類設備商，特別是在回檔時進場布局，目前的報酬率都相當不錯 。雖然這類標的高檔震盪大，但作為先進封裝與光通訊趨勢的受益者，其長線地位穩固 。

Apple (AAPL) 的現金流優勢：

蘋果在目前的 AI 競爭中，不需要像其他雲端巨頭 (CSP) 那樣投入極其龐大的基礎建設資源 。這使得蘋果的現金流維持在非常強勁的狀態，可能成為未來市場避險或回補的首選標的 。在大家都在燒錢拼 AI 的同時，蘋果這種「收租型」或「現金流型」的論述可能會重獲市場喜愛 。

3. 其他財經消息及知識 ​

CPU 缺貨可能比記憶體更嚴峻：

根據最新的報導與觀察，下半年 CPU 的供應緊俏程度可能超越記憶體，且可能面臨「有錢也買不到」的窘境 。記憶體在報價高時仍可買到，但 CPU 的產能若被 Agentic AI 的需求卡住，則會引發系統性的漲價 。這將對通用伺服器 (General Purpose Server) 的組裝成本產生巨大衝擊，是目前極需注意的供應鏈風險 。

Agentic AI 對 CPU 算力的重新定義：

Agentic AI 被認為是接下來 AI 應用的核心，而其運行邏輯與傳統 GPU 平行計算不同，更依賴 CPU 的處理能力 。市場目前對 GPU 的熱度極高，卻低估了 CPU 在這場 AI 革命中所扮演的「大腦與教練」角色 。隨著需求在下半年拉升，原本接近滿載的 CPU 產能可能會引發市場認錯並大幅上修估值 。

Uber 的 AI Token 額度提前用罄：

Uber 在開年後僅三四個月就用光了全年的 AI 額度，這被解讀為 AI 需求極其火熱的訊號 。然而主委認為這不全然是好事，關鍵在於這些 Token 的消耗是否有產出對應的績效與結果 。如果只是為了達到 KPI 而讓機器人空轉消耗 Token，最後會反映在難看的財務數字上 。

Token Maximization 轉向 Result Base：

市場目前的論述正從「你用了多少 AI」轉向「你透過 AI 賺了多少錢」 。過去只要公司改名或宣稱使用 AI 就會漲，但未來大眾會嚴格檢視其毛利與淨利是否有實質增加 。如果消耗大量 Token 卻沒有產出優質產品或提升營收，這類公司最終會被市場淘汰 。

傑文斯悖論在數位時代的應用：

雖然新技術提升了效率 (如 AI 減少單位算力需求)，但效率提升通常會誘發更廣泛的應用，導致總需求反而上升 。這在 AI 市場非常明顯，當 Token 變得便宜且好用時，大家會用得更多，進而帶動更多的硬體採購需求 。投資者不應擔心 AI 進步會減少對晶片的需求，事實通常恰好相反。

職業生涯後期的 Sales 化傾向：

奧特曼 (Sam Altman) 曾提到，任何職業發展到後期本質上都會變成 Sales，無論是工程師還是 CEO 。這意味著除了專業技術，溝通、管理、籌資以及向大眾「販售夢想」的能力將決定成功的上限 。目前的模型公司正大力招聘 Sales，正是為了將 AI 技術尋找到實質的商業出海口 。

4. Q&A ​

對已逝聽眾小徐的感性致詞：

宜蘭聽眾小徐不幸離世，主委應徐太太私訊請求在節目中向其道別，並祝願他跟著光走、不再疼痛 。主委向小徐保證徐太太非常勇敢，且主委會提供相關資源，絕不會讓他的孩子受凍挨餓 。他相信生命雖有終點，但能量與愛會延續，希望徐家能帶著這份寄託勇敢生活 。

成功不應僅以金錢來定義：

隨著年齡與財富累積，主委認為成功的定義已從「賺多少錢」轉向「健康的身體」與「良好的家庭關係」 。身邊有錢人很多，但能陪著孩子快樂長大、在 60 歲仍能進行高強度運動，才是真正的成功 。過度追求金錢有時只會帶來無謂的麻煩，心理的平靜與對家人的交代更為重要 。

退休資產配置與物慾控管：

對於擁有一億資產（含房產）想退休的聽眾，主委建議若能達到每年 5% 的報酬率，即有 500 萬可花用 。是否能退休取決於個人的物慾，若每年開銷僅 200 萬，現在即可退休；若需 400 萬則較緊繃 。退休後建議「退而不休」，去做一些有趣的小生意或兼差，維持生活的彈性與收入 。

台大兒童醫院護理人員的致敬：

有位護理師聽眾在台大兒科病房服務半年，主委對此感觸極深，因為他的次子也曾在台大兒一出生並受過專業照顧 。主委公開感謝所有在台大兒童醫院奮鬥的醫護人員，並認為那是一個讓奇蹟發生的地方 。他也鼓勵有餘裕的聽眾可以捐款支持，這是一件非常了不起的事 。

投資買「體驗」是最值得的花費：面對喜歡投資卻捨不得花錢的聽眾，主委指出買體驗的回憶是具備複利效應的，不應過度延遲享樂 。雖然投資者總會覺得錢不夠用、想多買幾張股票，但把錢分享給家人、帶他們去旅遊所獲得的幸福感是無價的 。每個人對價值的判斷不同，不需學別人買名表，只要自己花得爽且值得即可 。

處理育兒中的「兩顆球」趣事：

聽眾分享五歲兒子宣稱有兩顆拿不出來的球，結果是指自己的睪丸，引發爆笑 。主委藉此提醒，小孩在成長過程中會開始對身體產生好奇，父母應以此為契機進行性教育 。應教導孩子保護好這些重要的部位，不讓外人隨意觸摸，並隨時保持清潔以防感染 。
