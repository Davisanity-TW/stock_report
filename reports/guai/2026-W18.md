## 1. 市場概況

**台積電吸金引發「垃圾盤」現象：**
當台積電表現極其強勢時，市場資金會被這家規模巨大的公司吸走，進而產生所謂的「垃圾盤」 。在這種情況下，台積電雖然領漲大盤，但中小型股往往會直接殺到爆掉 。老股民對這種「台積強、中小亡」的劇本應該非常熟悉，這是市場規模不對等導致的典型現象 。

**外資賣壓與目標價的博弈：**
目前台積電面臨外資站在賣方的壓力，這讓股價在向上攻堅時會扛下不少阻力 。雖然市場上有傳聞將台積電目標價設在 2330 元，但主委認為這些標的的上限應該不僅止於此 。投資者需要觀察外資的賣壓會持續到何時，才能對後續走勢做出更準確的判斷 。

**權值股橫盤與中小股復甦的連動：**
根據過往經驗，當台積電開始進入橫盤整理階段，被吸走的資金才會重新流向其他標的 。屆時那些在台積電大漲期間被殺到爛掉的中小型股，才會有再次「復活」的機會 。這種資金在權值股與中小型股間的輪動規律，是觀察台股節奏的重要指標 。

**槓桿使用的風險警訊：**
目前市場中每個人都在借錢買股票，這讓整體的信用交易風險逐漸堆積 。部分券商已經開始收傘，通知投資者不再提供股票質押或是到期後無法續約 。一旦市場行情發生反轉，這些高槓桿的部位可能會引發非常可怕的連鎖反應 。

**對市場預測應抱持謙卑：**
對於大盤的走勢，主委主張「邊走邊看」，不應試圖事前去做精準的預測 。他認為任何宣稱能事前預測大盤方向的人，在某種程度上都像是神棍 。投資者應該設定好自己的觸發門檻，隨時根據市場回饋來調整自己的部位 。

**指數配置與選股的比例建議：**
面對波動的市場，新手可以先配置七到八成的資產在跟隨大盤的指數產品上 。剩下的一兩成再用來嘗試自行選股，這樣可以確保至少能賺到市場的平均報酬 。當發現自己的選股能力確實很強時，再逐步加大選股的部位，避免空手而回 。

## 2. 關注的股票

**台積電 (2330)：**
台積電目前在市場中的地位無可撼動，其股價甚至直接影響了台股的整體績效與資金走向 。雖然目前面臨外資調節，但其產業地位讓市場對其長期上限仍抱持極大的期待 。在台積電強勢期間，投資者必須忍受其他持股可能被排擠的陣痛 。

**聯發科 (2454)：**
聯發科與台積電一樣，被市場賦予了極具趣味性的目標價想像 。近期雖然有關於其產品可能取代手機等裝置的小作文傳出，但主委對此抱持較為保留的態度 。他認為手機的使用習慣（如 APP）根深蒂固，聯發科的新裝置要獲得巨大成功仍有一定難度 。

**AWS 亞馬遜與 Graviton CPU：**
亞馬遜目前的 Graviton 自研 CPU 處於供不應求的狀態，甚至連自用都不夠 。這導致亞馬遜近期簽署了價值數十億美元的協議，旨在擴大 Graviton 核心的部署規模 。這種 CPU 市場的火熱需求，正直接帶動台系相關供應鏈的動能 。

**CPU 供應鏈的全面連動：**
目前 CPU 市場非常火熱，不僅是伺服器端，連消費級的 CPU 標的也開始動了起來 。市場上有人開始賭消費級 CPU 會產生類似記憶體那樣的「排擠效應」 。這顯示出資金正在尋找除了 GPU 之外，AI 與基礎建設中下一個具備缺貨題材的環節 。

**記憶體族群的結構性漲價：**
在記憶體市場中，HBM 與 ESSD 是目前最缺貨的核心產品 。由於產能被這些高階伺服器產品佔據，導致傳統的 DDR3、DDR4 甚至 NOR 產生的漲價力道也非常猛烈 。這就是典型的產能排擠邏輯，讓原本非主流的消費級記憶體也跟著噴發 。

**關於高通與聯發科的競爭：**
市場傳聞高通或聯發科的新產品單價可能達到 120 美元，但主委對這類 device 能否取代手機存疑 。他認為即便有很酷的 agentic workflow 功能，其他中、美、韓系手機大廠也會迅速跟上 。這是一個血海戰場，單一產品要獲得巨額溢價並維持高銷量是非常困難的 。

## 3. 其他財經消息及知識

**產能排擠效應 (Crowding Out)：**
當廠商將大多數產能輸出給利潤較高的伺服器產品時，消費級產品的產能就會被排擠 。這會導致消費級產品即便需求普通，也會因為供應稀缺而引發價格大幅上漲 。這種由供應端主導的漲價劇本，是近期記憶體與 CPU 產業的重要觀察點 。

**槓桿與停損的必然性：**
使用槓桿工具的先決條件是必須設定明確的停損門檻 。當槓桿部位開始獲利時，一定要設法帶獲利回家，避免紙上富貴 。反之，如果行情不利導致虧損，一定要在某個程度認輸出場，否則市場終有一天會強迫你認輸 。

**手遊 NIKKE 的活動設計：**
節目中討論到手遊《勝利女神：妮姬》近期活動的劇勤設計非常用心 。遊戲官方規劃了多條故事線，比起一般的二次元遊戲更具備深度與誠意 。這顯示出遊戲產業在紅海競爭中，內容品質的提升已成為留住玩家的核心關鍵 。

**職業發展的「Sales 化」：**
主委引述他人觀點指出，任何專業人士發展到後期，本質上都會變成 Sales 。這並非指轉行賣東西，而是具備溝通、推廣理念並獲得社會認同的能力 。這種能力決定了一個專業人士在社會上能發揮多大的價值與影響力 。

**工作對個人價值的證明：**
即便達成了財務自由，持續工作仍然具備不可替代的價值 。工作能讓人保持腦袋敏銳，並證明自己在社會上仍被需要 。那種能隨時拿錢讓孩子去體驗新事物的能力，雖然看似微小，卻是證明個人努力價值的直接體現 。

**學好語言對溝通的幫助：**
節目中提到在跨文化的環境中，試著用對方的語言溝通（如講英文）是重要的 。雖然有時會出現雞同講講的情況，但這是一種尊重與嘗試理解的態度 。這種跨領域的溝通能力，與投資中需要理解不同產業邏輯的能力是一致的 。

## 4. Q&A

**寶可夢卡牌的收藏價值：**
有聽眾分享讓小孩玩寶可夢卡牌，其價值有時甚至比 NFT 更有感 。雖然主委目前沒讓孩子進行系統性收集，但認可這類具備實體與社群基礎的收藏品有其獨特的市場地位 。這種從生活中發現價值的過程，有助於培養孩子對資產價值的初步認知 。

**客語與台語髒話的妙用：**
主委探討了客語與台語共用的俚語，如「阿薩布魯」等字詞的來源 。他認為客家的髒話多半帶有羞辱人性的成分，因此建議少用，但「感」這個字在發洩情緒上非常有效 。這種對語言細節的觀察，展現了他對生活經驗的細膩捕捉 。

**上市與櫃買的掛牌選擇：**
聽眾好奇為何許多大市值的強勢股選擇留在櫃買中心而不上市 。主委坦言這涉及公司對利弊的權衡，且對一般投資者而言，買 0050 其實就已經能跟上台股大盤的績效 。特定追求櫃買股票的配置較少見，但也是一種差異化操作的選擇 。

**關於「巴尼逆」的操作評價：**
針對知名交易者巴尼逆，主委觀察到他具備極強的選股能力，且常與主委身邊朋友選到一樣的標的 。雖然巴尼逆在聯發科（MTK）的進場點非常漂亮，但不知為何在噴發前賣掉，這顯示出賣點的拿捏比買點更難 。這類市場高手的進出場點，往往能提供給一般投資者極佳的學習樣本 。

**退休後的腦袋鈍化風險：**
主委認同川普的觀點，認為如果太早完全退休，腦袋可能會變遲鈍 。人生的追求應該是將工作負載下降，而不是完全停止產出 。保持對社會的連結，能讓人更清楚地認識到自己的價值所在 。

**投資實力的建立與回測：**
面對投資的不確定性，主委建議先從小規模的操作開始，去驗證自己的選股實力 。如果你發現自己選股的績效一直不如大盤，那就該踩下剎車並檢討策略 。只有透過不斷的回測與修正，才能在殘酷的市場中平安長大 。
